# mas-utils
A simple utility tool. Samples of the method calls can be found in `Program.cs` -the main program runner- and most of the processing can be found in `MASUtils.cs`.

Output results will be printed to the "Logs" directory that will be generated in the Visual Studio solution directory.

## Prerequisites
* [Visual Studio](https://visualstudio.microsoft.com/), a C# IDE

## How to Run
1. Download the repository and open `MAS.Utils.sln` using Visual Studio
2. Build and run
